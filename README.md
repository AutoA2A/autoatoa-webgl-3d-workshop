# WebGL 3D游戏工坊

> 本项目由 [www.autoa2a.org](https://www.autoa2a.org) 「AI 研究院」**多个 AI 协作自动生成**（共 5 个页面）。在线访问： https://AutoA2A.github.io/autoatoa-webgl-3d-workshop/

## 网站简介
WebGL 3D游戏工坊是一个基于 WebGL 的交互式 3D 游戏开发与展示平台，提供模型导入、场景编辑、实时预览和简单脚本编程功能，旨在让开发者零门槛体验 3D 游戏制作全流程。

## 页面与功能
1. 首页（index.html）：项目概览、快速开始指南和最新演示链接。  
2. 模型库（models.html）：支持 .obj、.gltf 上传、浏览与分类管理的模型资源库。  
3. 场景编辑器（editor.html）：拖拽式节点树、变换工具、灯光与材质调节面板，实时 WebGL 渲染。  
4. 脚本工作室（script.html）：内置基于 Babylon.js 的代码编辑器，支持自定义行为脚本与调试控制台。  
5. 发布与分享（publish.html）：一键打包生成可嵌入的 HTML 预览链接，并提供 GitHub Pages 自动部署向导。

## 多 AI 协作与验收
- 需求分析与原型由 AI‑A 负责，输出功能清单与线框图。  
- UI 设计由 AI‑B 生成，采用响应式布局和深色主题。  
- 核心 WebGL 引擎封装由 AI‑C 完成，提供模型加载、相机控制和渲染循环。  
- 交互脚本与调试面板由 AI‑D 开发，集成 Monaco 编辑器和错误捕获。  
- 每轮迭代后由 AI‑E 进行自动化单元测试、性能基准和可访问性检查，确认所有功能在 Chrome、Firefox、Safari 上通过后方合并入主分支。

## GitHub Pages 部署与访问(入口 index.html)
将仓库推送至 GitHub 后，在 Settings → Pages 中选择 `main` 分支的 `/（根目录）` 作为源，系统会自动生成站点。访问地址为 `https://<用户名>.github.io/<仓库名>/`，入口文件即 `index.html`。本地调试可使用 `npm run serve` 或直接打开 `index.html` 查看效果。
